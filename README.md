# Common Lisp Incognia APIs Wrapper

Simple as:

```lisp
(authenticate '("client-id" . "client-secret")) ;; or create a credentials.yml file

(incognia:signups :installation-id "your-installation-id"
                  :address-line "your-address-line")
```
